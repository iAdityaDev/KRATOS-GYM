First-time setup

This app uses Supabase as its database (free, backed up automatically).

Steps:
1. Go to supabase.com → create a free account
2. Create a new project
3. Go to Settings → API
4. Copy your Project URL and anon public key below
5. Go to SQL Editor and run the SQL shown below
Already have a project running?
Just run these lines in your existing Supabase SQL Editor, then reload the app — no need to redo setup: 
alter table members add column if not exists is_archived boolean default false;
alter table settings add column if not exists gym_photo text;
Run this SQL in Supabase SQL Editor: 
create table members (
  id uuid default gen_random_uuid() primary key,
  name text not null,
  phone text,
  address text,
  photo_url text,
  plan_name text,
  fees_paid numeric,
  days int,
  start_date date,
  end_date date,
  is_draft boolean default false,
  is_archived boolean default false,
  notes text[],
  note_dates text[],
  created_at timestamptz default now()
);
alter table members enable row level security;
create policy "allow all" on members for all using (true);

create table settings (
  id int primary key default 1,
  password text,
  plans jsonb,
  gym_photo text
);
